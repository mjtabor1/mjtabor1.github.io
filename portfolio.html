About me

One of the first programs I wrote was a game that asked two users for a number and compared those numbers to a random number generator. Whoever guessed closest to the random number was the winner. Simple, I know but ever since then I've been hooked. It was very surreal to me that I could type in some code on a screen and create a usable, working program that another human being could use. I hope to one day be a master at my craft and develop something that will help change the world for the better.
